網頁設計入門

行前準備

  • 瀏覽器：下載並安裝 Google Chrome
  • 通訊軟體：使用 Google Hangouts，若沒有 Gmail 帳號，請點此註冊。
  • 文字編輯器：下載並安裝 Visual Studio Code
  • 發佈工具：下載並安裝 Github Desktop，若沒有 Github 帳號，請點此註冊

你將學會

  • 解構網頁的介面
  • 使用 HTML 常見的標籤寫出個人網頁
  • 發佈網站至 Github Pages
  • 修改並發佈從網路下載的炫砲版型

  2. Github Pages 入門

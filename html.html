一、我的第一個網頁

  1. 在 VS Code 新增檔案，命名為 index.html
  2. 在 index.html 中輸入任意文字
  3. 雙擊檔案，以瀏覽器開啟 index.html

二、解構網頁的介面

  1. 選擇一個你經常瀏覽的網頁（想像你要寫出一模一樣的網頁）
  2. 從上至下、左至右分解網頁的結構

三、HTML 語法

  1. 網頁是由元素所組成
  2. 元素是由開始標籤、內容、結束標籤所組成。e.g. <h1>我是內容</h1>
  3. 內容可以是元素，稱為巢狀元素。e.g. <h1><b>我</b>是內容</h1>
  4. 沒有內容的元素則可以省略結束標籤。e.g. <hr>
  5. 在開始標籤加上 attribute="value"，可以為元素附上值為 value 的 attribute。e.g. <img src="cat.jpg">

小練習

使用 Chrome 開發者工具去觀察網頁的原始碼
